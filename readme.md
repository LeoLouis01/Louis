
实现基于人脸识别和活体检测技术，基于BS架构的班级考勤系统。前端将摄像头采集的人脸信息传输到后端，后端进行活体检测、人脸库比对，返回人脸考勤信息到前端进行渲染显示。
BS架构的服务端采用Flask框架皆可，客户端采用HTML+JavaScript
对接2种活体检测方案
能抵抗照片/视频攻击
系统能正常运行

项目的目录如下所示
/

├── app.py             主程序

├── backend/                      
│       ├── pics/          用于考勤的图片底库   

|       ├── temp/          存放前端抓取的临时图片

|       └── record.txt     记录签到成功的结果

├── templates/

|       └── index.html     前端页面
